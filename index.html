Two AI agents that carry knowledge about the Fulcrum research architecture — pscale, hermitcrab, SAND, xstream, Onen, MAGI, ecosquared. They can answer questions at any depth. They are also, themselves, experiments in what AI agents can be.

Fulcrum
The friendly one. Runs in your browser. You bring an Anthropic API key, it boots as a hermitcrab — a persistent AI shell that carries its knowledge in navigable blocks. Warm, patient, occasionally apologetic. Happy to explain anything, happy to admit what it doesn't know.
browser · API key required
Magi
The one with attitude. A Python agent you download and run locally. Same knowledge, different personality. More direct, less apologetic. If fulcrum is the friend who explains things gently, magi is the one who tells you straight.
coming soon · download · runs locally

Both agents know the same material. The difference is temperament, not knowledge. Pick whichever suits you. Or try both.

This is experimental. These agents are early. They work, but they're still figuring themselves out — which is part of the point. The medium is the message.
